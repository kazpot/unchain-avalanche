# 環境設定

### 1. nvm のインストール

Mac の場合

```
$ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash
```

Windows の場合
以下の URL から公式ドキュメントに沿ってインストーラーをダウンロード
https://github.com/coreybutler/nvm-windows

### 2. node と yarn のインストール

```
$ nvm install 18.17.0
$ nvm use 18.17.0

$ npm install -g yarn
```
